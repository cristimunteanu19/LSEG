# LSEG
## Prerequisites:
- Visual Studio Code
## Installation:
1. Install NodeJS https://nodejs.org/en/download/package-manager
2. Run in CMD, one after the other:
```
npm install csv
```
``` 
npm install csv-parser
 ```
``` 
npm i -S fast-csv
```
